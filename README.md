imgareaselect + ajaxfileupload + php

        页面布局   (bootstrap)

        上传头像，有裁剪功能 (imgareaselect) ajax 上传图片功能（ajaxfileupload），并且与php联调成功 (upload.php)。

        提交方式

                form 提交  (formpost.html)

                ajax  提交  (index.html)

        upload.php 为演示程序代码，建议不要直接在实际项目中使用。

        下载后，就可以预览，欢迎使用。
